# laraflat
Advanced laravel system to build admin panel 


## Download the files 

download all files

## Requirements

PHP >= 5.6.4 , 
PHP Curl extension 

## Install  The Dependencies

now type this line on your console

```
composer install
```
```
php artisan key:generate
```

## migrate

```
  php artisan migrate
```

## Seed Database 

```
  php artisan db:seed
```

## Start server


```
  php artisan serve
```

## Login

```
  email : admin@gmail.com
  pass : admin
```

## Go to admin path

```
  http://127.0.0.1:8000/admin/home
```

## Add New Model

```
    php artisan make:admin_model NameOfModel
```
